# adamnamrouchi.com

Personal portfolio website built with modern web technologies.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Animations:** Framer Motion, GSAP
- **Physics:** Matter.js
- **WebGL:** OGL
- **Smooth Scroll:** Lenis
- **Font:** Gilroy

## Features

- Animated text and background effects
- Project showcase
- Experience timeline
- Hackathons section
- Contact page

## Getting Started
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.
